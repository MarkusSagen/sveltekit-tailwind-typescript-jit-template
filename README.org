#+title: Sveltekit app with TailwindCSS, Vite and Typescript


** Setting up
   #+BEGIN_SRC shell
   NAME="sveltekit-vite-tailwind-app"
   npm init svelte@next $NAME 
   # Choose your options 

   cd $NAME
   npm install
   git init && git add -A && git commit -m "Initial commit"
   npm run dev -- --open
   #+END_SRC
